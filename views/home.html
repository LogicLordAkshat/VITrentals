About VITrentals

VITrentals is a comprehensive car rental management system developed as a final project for the Database Management Systems (DBMS) course at VIT University.

This project demonstrates advanced database concepts including normalization up to BCNF, complex SQL queries, CRUD operations, and full-stack web development using Node.js, Express.js, MySQL, and modern frontend technologies.
